# Plantilla modular en LaTeX para libro técnico

Proyecto preparado para **Overleaf** y organizado por módulos.

## Compilación
- Compilador recomendado: **pdfLaTeX**
- Archivo principal: **main.tex**

## Estructura
- `config/` configuración global del proyecto
- `frontmatter/` portada, portadilla, créditos, resumen, abstract, introducción, etc.
- `capitulos/` capítulos del libro
- `backmatter/` referencias, autores, anexos
- `figuras/`, `tablas/`, `anexos/` carpetas para contenido gráfico y material complementario

## Qué debes editar primero
1. `config/metadata.tex`
2. `frontmatter/resumen.tex`
3. `frontmatter/abstract.tex`
4. archivos dentro de `capitulos/`

## Nota
La plantilla deja espacios para:
- ISBN
- DOI
- CIP / ficha catalográfica
- datos editoriales finales
- autores y afiliaciones

Estos campos deben completarse cuando la editorial los confirme.
